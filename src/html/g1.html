auto-fit
自适应列的数量，栅格将会根据容器的宽度调整其数量。

minmax(100px, 1fr)代表每列的列宽将至少为 100px
如果有更多的可用空间
栅格布局会将其均分给每列

.container9 {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
    grid-template-rows: repeat(2, 100px);
    gap: 10px 20px;
}
